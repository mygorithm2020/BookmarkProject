- 기간 : 2024. 06. 06 ~  2024. 08. 13
- 기술 스택 : HTML, CSS, JavaScript, NodeJS, NestJS, MySQL
- 목표 : v1 버전(정적웹사이트)에 백엔드 서버 및 DB 추가
